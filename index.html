Register Wajah (Single ID)

Website ini menangkap foto dari webcam dan mengirimkan satu foto untuk daftar wajah. ID yang tersimpan bersifat tetap (single user).

Status: Idle

ID akan dikirim bersama foto. Untuk sistem single-user biarkan tetap.

Ganti dengan URL ngrok kamu setelah menjalankan: ngrok http 5000

Catatan cepat:
  • Tekan Start Camera untuk mengaktifkan webcam.
  • Gunakan Ambil Foto untuk memotret, lalu klik Register untuk mengirim satu foto.
  • Endpoint default menunjuk ke server lokal; jika menggunakan ngrok, ganti field Endpoint dengan URL ngrok.
